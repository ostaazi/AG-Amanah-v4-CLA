# Release Prep Checklist

## الحالة الحالية
- تم استكمال ملفات الاستعادة الرئيسية (UI + Services + Workers).
- تم ربط المسارات الأساسية داخل `App.tsx` مع `feature flags`.
- تم توسيع اختبارات الانحدار وتشغيلها بنجاح.

## ما تم التحقق منه
- جميع الملفات المخططة في الخطة التفصيلية موجودة فعليًا.
- البناء ينجح: `npm run build`.
- الاختبارات تنجح: `npm run test:run`.

## المتبقي قبل الإقفال النهائي
1. تنفيذ smoke test يدوي لمسارات التشغيل الحرجة:
   - `/command-center`
   - `/dev-resolution`
   - `/parent-ops`
   - `/playbooks`
   - `/war-room`
2. مراجعة نصوص الواجهة القديمة التي تحتوي تشويش ترميز في بعض الملفات التاريخية (إن ظهرت بالمشاهدة).
3. تنظيف ملفات النسخ المؤقتة غير اللازمة في جولة لاحقة بعد تثبيت الإصدار.

## قرار المرحلة
- الانتقال إلى مرحلة hardening النهائي مع إبقاء النسخ الاحتياطية بعد كل جزء.
