
# Amanah AI: Advanced Parental Shield
# أمانة: نظام الرقابة الأبوية المتقدم بالذكاء الاصطناعي

---

## Arabic / اللغة العربية

### نظرة عامة
نظام "أمانة" هو منصة أمنية وتربوية متكاملة تعتمد على تقنيات الذكاء الاصطناعي لتوفير بيئة رقمية آمنة للأطفال. يتميز النظام بقدرته على التحليل الاستباقي للمحتوى واكتشاف التهديدات قبل وقوع الضرر، مع توفير أدوات تحليل نفسي وجنائي متقدمة لمساعدة أولياء الأمور.

### الميزات الرئيسية
*   التحليل السلوكي المتقدم: مراقبة الأنماط اللغوية والبصرية لاكتشاف التنمر، الابتزاز، ومحاولات الاستدراج.
*   الخزنة الجنائية الرقمية: أرشفة مشفرة للأدلة والرسائل المشبوهة لاستخدامها في الإجراءات الرسمية.
*   نظام النبض النفسي: تقارير دورية حول الحالة النفسية للطفل بناءً على تفاعلاته الرقمية.
*   التحكم العتادي المباشر: القدرة على قفل الكاميرا، الميكروفون، أو تعتيم الشاشة فورياً في حالات الطوارئ.
*   المستشار التربوي الذكي: واجهة تفاعلية مدعومة بنماذج Gemini لتقديم نصائح تربوية متخصصة.

### البنية التقنية
*   واجهة المستخدم: React.js مع نظام Tailwind CSS للتصميم المستجيب.
*   محرك الذكاء الاصطناعي: Google Gemini API للتحليل اللغوي العميق.
*   المعالجة المحلية: TensorFlow.js للكشف البصري الفوري على الجهاز.
*   قاعدة البيانات والتحقق: Firebase Cloud Firestore و Firebase Authentication.
*   التشفير: بروتوكول AES-GCM 256-bit لتأمين البيانات المحلية.

### طريقة التثبيت
1. قم بتحميل ملفات المشروع.
2. قم بتثبيت المكتبات اللازمة باستخدام الأمر: `npm install`.
3. قم بإعداد متغيرات البيئة وإضافة مفتاح Gemini API في ملف `API_KEY`.
4. ابدأ التشغيل عبر الأمر: `npm start`.

## English / اللغة الإنجليزية

### Overview
Amanah AI is a comprehensive security and educational platform that leverages artificial intelligence to provide a safe digital environment for children. The system features proactive content analysis and threat detection capabilities, integrated with advanced psychological and forensic tools to assist parents.

### Key Features
*   Advanced Behavioral Analysis: Monitoring linguistic and visual patterns to detect bullying, blackmail, and predatory attempts.
*   Digital Forensic Vault: Encrypted archiving of evidence and suspicious messages for official proceedings.
*   Psychological Pulse System: Periodic reports on the child’s emotional state based on their digital interactions.
*   Direct Hardware Control: Instant ability to lock the camera, microphone, or dim the screen during emergencies.
*   Smart Parental Advisor: An interactive interface powered by Gemini models to provide specialized parenting advice.

### Technical Stack
*   Frontend: React.js with Tailwind CSS for responsive design.
*   AI Engine: Google Gemini API for deep linguistic analysis.
*   Local Processing: TensorFlow.js for real-time, on-device visual detection.
*   Database & Auth: Firebase Cloud Firestore and Firebase Authentication.
*   Encryption: AES-GCM 256-bit protocol for securing data at rest.

### Installation
1. Clone the repository.
2. Install dependencies using: `npm install`.
3. Set up environment variables and add your Gemini API key in the `API_KEY` field.
4. Run the application using: `npm start`.

---

### Security & Privacy
This project is built with a Zero-Knowledge Architecture. All sensitive child data is encrypted locally before being stored, ensuring that only authorized parents have access to the forensic logs.
