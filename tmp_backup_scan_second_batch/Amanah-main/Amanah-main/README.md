
# Amanah AI: Advanced Parental Shield and Behavioral Intelligence
# أمانة: نظام الرقابة الأبوية المتقدم والذكاء السلوكي

---

## English Documentation

### Overview
Amanah AI is a high-integrity security platform designed for modern digital parenting. It integrates deep linguistic analysis, real-time computer vision, and forensic logging to provide a proactive safety net for children across social and gaming platforms. The system is built on a foundation of privacy-first architecture, ensuring that sensitive data remains encrypted and accessible only to authorized guardians.

### Core Technical Pillars
1. **Hybrid Threat Detection**: Combines on-device (Edge) processing for zero-latency response with cloud-based generative AI (Gemini) for complex semantic analysis.
2. **VisionGuard Protocol**: Real-time visual monitoring using local neural networks to detect sensitive or dangerous content without transmitting raw images to the cloud.
3. **Forensic Evidence Vault**: A tamper-proof repository for digital evidence, including chat logs and timestamps, formatted for professional or legal review if necessary.
4. **Behavioral Pulse**: A sophisticated analytics engine that monitors emotional trends and anxiety levels based on digital interaction patterns.

### Deployment on Vercel
To deploy this application on Vercel:
1. Push the repository to GitHub, GitLab, or Bitbucket.
2. Import the project into the Vercel Dashboard.
3. Configure the `API_KEY` environment variable with your Google Gemini API key.
4. Vercel will automatically detect the configuration and deploy the application as a single-page application (SPA).

### Technical Stack
* **Runtime**: React 19.x with ESM modules.
* **Styles**: Tailwind CSS for high-performance responsive UI.
* **Intelligence**: Google Gemini Pro/Flash API.
* **On-Device ML**: TensorFlow.js and NSFWJS.
* **Persistence**: Firebase Firestore and Encrypted IndexedDB.

## التوثيق باللغة العربية

### نظرة عامة
منصة "أمانة" هي نظام أمني متطور مصمم للرقابة الأبوية الحديثة. يدمج النظام بين التحليل اللغوي العميق، والرؤية الحاسوبية الفورية، والتدوين الجنائي لتوفير شبكة أمان استباقية للأطفال عبر منصات التواصل والألعاب. يعتمد النظام على بنية تحتية تولي الخصوصية الأولوية القصوى، مما يضمن بقاء البيانات الحساسة مشفرة ولا يمكن الوصول إليها إلا من قبل الأوصياء المعتمدين.

### الركائز التقنية الأساسية
1. **الكشف الهجين عن التهديدات**: يجمع بين المعالجة المحلية (Edge) للاستجابة السريعة وبين الذكاء الاصطناعي السحابي (Gemini) للتحليل الدلالي المعقد.
2. **بروتوكول VisionGuard**: مراقبة بصرية حية باستخدام شبكات عصبية محلية للكشف عن المحتوى الحساس دون إرسال الصور الخام إلى السحابة.
3. **خزنة الأدلة الجنائية**: مستودع مؤمن ضد التلاعب للأدلة الرقمية، يشمل سجلات المحادثات والأختام الزمنية، بصيغة معدة للمراجعة المهنية أو القانونية عند الضرورة.
4. **النبض السلوكي**: محرك تحليل متقدم يراقب الاتجاهات العاطفية ومستويات القلق بناءً على أنماط التفاعل الرقمي.
